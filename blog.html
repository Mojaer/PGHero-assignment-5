Question: What is DOM and what is the purpose of DOM?

Answer: DOM is The Document Object Model. it represents the HTML interface so that Javascript can interact with the page to change the style and elements in it.

Question: How will you select HTML elements using DOM? Name the DOM methods?

Answer: We can select HTML elements using DOM by DOM methods. There are five DOM methods:

  1. getElementsByClassName()
  2. getElementById()
  3. getElementsByTagName()
  4. querySelectorAll()
  5. querySelector()

Question: What is event bubble?

Answer: event bubble is one kind of character of DOM which indicates that if any child element is selected the event propagates to the parent element

Question: What is a callback function and why will you use it?

Answer: The callback function is function which is used as an argument of another function. This function is used to do complex task where the execution of the a function can wait until a specific event occurs
